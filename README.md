# Deadlock Simulation (NRP 152024165)

## Deskripsi
Program ini mensimulasikan kondisi deadlock menggunakan Python threading.

## Penjelasan
- Mahasiswa A memegang printer dan menunggu scanner
- Mahasiswa B memegang scanner dan menunggu printer
- Terjadi deadlock karena kedua proses saling menunggu

## Cara Menjalankan
python 152024165_DeadLock.py

## Hasil
Program akan berhenti (freeze) karena deadlock
